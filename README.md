# aak-server-account

## 项目简介

aak 账号服务端，基于 Spring Boot 3 搭建

## 环境配置

| 工具      | 版本       |
|---------|----------|
| Java    | 17.0.5   |
| Gradle  | 7.6      |
| GraalVM | 22.3.0   |
| IDEA    | 2022.3.1 |

### 邮箱密码登录

### 邮箱注册

- 邮件激活链接

### 密码找回

- 邮箱重置密码链接

### 账号注销

- 邮箱确认注销链接

### 个人信息

- 获取
- 修改

### 用户数据

- 获取/初始化
- 删除/初始化
- 修改
